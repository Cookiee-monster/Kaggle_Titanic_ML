1. Wstęp¶

Celem pracy byo sprostaniu oglnodostpnemu wyzwaniu zamieszczonym na portalu Kaggle. Temat wyzwania dotyczy problemu klasyfikacji binarnej czy dany pasaer przey katastrof statku Titanic w 1912 roku.

Zbiorem treningowym wraz z ze zmienn celu by zbir zawierajcy rekordy dla 819 pasaerw wraz z danymi zawartymi w 12 kolumnach

Sowniczek zmiennych:

Zmienna Definicja Klucz
Survived Czy dany pasaer przey 0 = Nie, 1 = Tak
Pclass Klasa biletu 1 = 1wsza, 2 = 2ga, 3 = 3cia
Sex Pe
Name Imi i nazwisko pasaera
Age Wiek w latach uamek gdy mniejszy ni 1 rok, w formacie xx.5 gdy
wiek by estymowany
SibSp liczba rodzestwa/maonkw na pokadzie Sibling(rodzestwo) = brat, siostra, brat przyrodni, siostra
przyrodnia
Spouse (maonkowie) = m, ona
(kochanki i narzeczone nie byy uznawane)
Parch Liczba rodzicw/dzieci na pokadzie Parent (Rodzic) = mama, tata

Dziecko = crka, syn, przybrana crka, przybrany syn

Niektre dzieci podroway tylko z niani - wtedy Parch = 0
Ticket Numer biletu
Fare Koszt biletu
Cabin Numer kabiny
Embarked Port zaokrtowania C = Cherbourg, Q = Queenstown, S = Southampton

5. Modelowanie¶

Zdecydowano si na wykorzystanie trzech rodzajw modelw:

  • regresji logistycznej poprzedzonej normalizacj wedug metody MinMax
  • metody lasw losowych - Random Forest
  • metody xGBoost

Dodatkowo w celu kalibracji hiperparametrw zdecydowano si na wykorzystanie metody GridSearch dla kadego z estymatorw by znale najlepszy rezultat dla metryki AUC (area under ROC curve) w procesie Cross Validation dla 5 prb (foldw).

Ponadto sprawdzono podejcie z jednym modelem dla caego zbioru oraz uczenie oparte na osobnych modelach dla zbiorw podzielonych pod ktem pci.

Najlepsze z modeli dla danego typu estymatora poddano finalnej walidacji krzyowaej z wykorzystaniem 5 prb i metryk:

  • F1
  • Accuracy
  • Precision

Regresja liniowa z normalizacją¶

W procesie Grid Search poszukiwano optymalnego zestawu parametrw dla:

  • wartoci C - odpowiadajacej regularyzacji (odwrotno parametru lambda)
  • metodzie regularyzacji (penalty) - wedug metody l1 (Lasso) i l2 (Ridge)

def preapre_data_predict(df):
    
    """
    Funkcja odpowiada za przygotowanie zbioru testowego do postaci umoliwiajcej predykcj 
    z wykorzystaniem finalnego modelu
    
    Parameters:
    df : DataFrame
        Obiekt wyjciowy zawierajcy zbir testowy
        
    Return:
    X_test : DataFrame
        Obiekt stanowicy zbir testowy przeksztacony do postaci wykorzystywanej w modelu
    """
    
    # Wykorzystanie "PassengerId" jako indeksu zbioru
    df.set_index("PassengerId", inplace=True)

    # Wyrzucenie zmiennej Ticket
    df.drop(["Ticket", "Cabin"], axis=1, inplace=True)

    # Stworzenie zmiennej Solo dla osb podrujcych w pojedynk
    df["Solo"] = ((df["SibSp"] == 0) & (df["Parch"] == 0)).astype("category")

    # Stworzenie zmiennej Crew dla osb podrujcych za darmo
    df["Crew"] = df["Fare"] == 0

    # Zmiana zmiennej "Sex" na zmienn binarn "Female"
    df.rename(columns={"Sex": "Female"}, inplace=True)
    df["Female"] = df["Female"].replace(to_replace={"female": 1, "male": 0})

    # Binaryzacja zmiennych "Crew" oraz "Solo"
    for col in ["Crew", "Solo"]:
        df[col] = pd.factorize(df[col])[0]
    
    # Uzyskanie zmiennej "Title" wykorzystujc zmienn "Name"
    df["Title"] = df["Name"].str.extract("(\w*\.)")
    
    # Wyodrbnienie kategorii "Other" dla zmiennej "Title" w przypadku gdy tytu nie naley do listy
    # "Mrs.", "Miss.", "Mr.", "Master."

    # Zamiana typu zmiennej na object
    df["Title"] = df["Title"].astype("object")

    # Ustawienie nowej wartoci dla tytuw nie nalecych do listy
    df.loc[~df["Title"].isin(["Mrs.", "Miss.", "Mr.", "Master."]), "Title"] = "Other"

    # Powrt do typu zmiennej bdcej kategori
    df["Title"] = df["Title"].astype("category")

    # Zmiana zmiennej "Title" na zmienne binarne
    dummy_title = pd.get_dummies(df["Title"], prefix="Title")

    # Odrzucenie zmiennej "Title_Other" - One-hot Encoding
    dummy_title.drop("Title_Other", axis=1, inplace=True)
    # Dodanie nowych kolumn do istniejcego zbioru
    df = pd.concat((df, dummy_title), axis=1)

    
    # Zmiana zmiennej "Embarked" na zmienn "Embarked_Q" wskazujc czy dany pasaer okrtowae si w Queenstown
    df.rename(columns={"Embarked": "Embarked_Q"}, inplace=True)
    df["Embarked_Q"] = df["Embarked_Q"].replace(to_replace={"Q": 1, "S": 0, "C": 0})

    # Dodanie zmiennej bdcej kwadratem wieku
    df["Age_2"] = df["Age"] ** 2

    # Odrzucenie zmiennych "Name" oraz "Title"
    df.drop(["Title", "Name"], axis=1, inplace=True)
     
    # Zmiana typu danych dla zmiennej Pclass
    df["Pclass"] = df["Pclass"].astype(int)
    
    # Deep copy df do zmiennej X_test
    X_test = df.copy()

    return X_test
